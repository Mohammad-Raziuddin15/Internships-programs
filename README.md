# 🎓 Internship Programs Website

A responsive Internship Programs website built using HTML and CSS. This project showcases different internship opportunities in a clean card-based layout with modern styling and responsive design.

## 📌 Project Overview

The Internship Programs Website provides information about various internship domains, including Web Development, App Development, Python Programming, and Java Programming. Each internship card displays key details such as duration, location, stipend, and start date, along with a link to learn more.
